# shinyKeyboardShortcuts
Keyboard Shortcuts for Shiny Apps

Just an experiment

Install with:

```
devtools::install_github("ebailey78/shinyKeyboardShortcuts")
```

Run the example with:
```
library(shinyKeyboardShortcuts)
runExampleSKS("old_faithful")
```
